List Categories

GET https://cagkquiz.herokuapp.com/restapi.php?api=categories

Parameters

ParameterValueDescriptionData Type
sortasc | ascOrder sort attribute ascending or descendingString
orderbycat-id |
category-name |
category-image |
random		Sort collection by object attributeString
cat-idLimit result set to items with more specific by `cat_id`Integer
category-nameLimit result set to items with more specific by `category_name`String
category-imageLimit result set to items with more specific by `category_image`String

Example

- https://cagkquiz.herokuapp.com/restapi.php?api=categories
- https://cagkquiz.herokuapp.com/restapi.php?api=categories&orderby=cat_id
- https://cagkquiz.herokuapp.com/restapi.php?api=categories&orderby=random
- https://cagkquiz.herokuapp.com/restapi.php?api=categories&orderby=cat_id&sort=desc



Retrieve a Category 

GET https://cagkquiz.herokuapp.com/restapi.php?api=categories&cat-id=<cat-id>

Parameters

ParameterValueDescriptionData Type
cat-idUnique identifier for the object and use random for random IDsInteger

Example

- https://cagkquiz.herokuapp.com/restapi.php?api=categories&cat-id=1
- https://cagkquiz.herokuapp.com/restapi.php?api=categories&cat-id=2
- https://cagkquiz.herokuapp.com/restapi.php?api=categories&cat-id=random
